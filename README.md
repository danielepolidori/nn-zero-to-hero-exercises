# "Neural Networks: Zero to Hero" exercises

This repository contains my personal solutions to the exercises covered in Andrej Karpathy's "[Neural Networks: Zero to Hero](https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ)" playlist, completed as part of the Artificial Intelligence course, during my Master's degree in Computer Science at the University of Bologna.

To explore the benefits of attention mechanisms, the tutorial follows six videos from the playlist — from the second to the seventh — progressively building toward the Transformer architecture.  
The objective is to generate new personal names from a list of approximately 32,000 existing ones (including _Emma_, _Olivia_, _Ava_, _Isabella_, and _Sophia_).
The problem is approached through several different methods, each seeking a progressively better solution. The best results are achieved using the Transformer architecture and, consequently, attention mechanisms.
